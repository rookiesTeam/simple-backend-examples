# NodeJS GuestBook API Server

Simple server featuring a CRUD Rest API Guestbook to be used by some frontend to-be-developed.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

```
git clone https://github.com/rookiesTeam/simple-backend-examples.git
cd simple-backend-examples
git fetch
git checkout nodejs-guestbook
```

### Prerequisites

* Node v6.9.1
* npm 3.10.8

### Installing

After cloning the repository, installation can be made using these commands:

```
cd simple-backend-examples/backend/nodejs/
npm install
```

## Deployment

```
cd simple-backend-examples/backend/nodejs/
npm start
```
